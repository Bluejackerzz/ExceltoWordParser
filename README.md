Excel to Word Parser
this is my own project that i create, to boost productivity for a case where theres a Document that needs to be filled from Excel to Word.
instead of fill it one by one, by hands-by seeing all of those Buzzy-Eyed countless list of documents, you can just do it by automatically
using this parser.


Easy to use, Compatible with .XLSX and can be use with all purpose of excel layout
